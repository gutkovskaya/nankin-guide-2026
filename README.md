# Путеводитель «Нанкин 2026» — сайт

Статический сайт-путеводитель, который публикуется на Netlify
(**nankin-guide-2026.netlify.app**) через авто-деплой из этого репозитория.

- `index.html` — сам путеводитель (свежая версия).
- `img/` — карты, схема метро, иконки приложений, логотипы брендов, фото.
- `netlify.toml` — настройки публикации (корень репозитория, без сборки).

## Как обновить сайт

1. Заменить `index.html` (и при необходимости файлы в `img/`) свежей версией.
2. `git add -A && git commit -m "обновление путеводителя" && git push`
3. Netlify сам пересоберёт и опубликует — через ~20–30 секунд.

> Источник правды для текста путеводителя — основной репозиторий
> (`Start/travel/Нанкин 2026/Путеводитель.html`). Сюда кладётся его копия как `index.html`.
